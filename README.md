# Cross Thread Queue

## Introduction 
A thread safe queue that can be used in multi-thread project

## Tested Environment
- Ubuntu 18.04
- g++ 9.4.0
- cmake 3.10.2

## How to Build example
```bash
rm build -rf
mkdir build && cd build
cmake -DCMAKE_INSTALL_PREFIX=./ ..
make install/strip
```

## How to Run example
```bash
./lib/ctQueue
```
